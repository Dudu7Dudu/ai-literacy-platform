- AI წიგნიერების სასწავლო პლატფორმა

-- პროექტის აღწერა

AI წიგნიერების სასწავლო პლატფორმა წარმოადგენს ვებდაფუძნებულ საგანმანათლებლო სისტემას, რომელიც შექმნილია ხელოვნური ინტელექტის შესახებ საბაზისო ცოდნის გავრცელებისა და შეფასებისთვის.

პლატფორმა მომხმარებლებს სთავაზობს:

* სასწავლო მოდულებს;
* გაკვეთილებს;
* ქვიზებს;
* Branching Scenario ტიპის ინტერაქტიულ შეფასებებს;
* ადმინისტრაციულ პანელს კონტენტის სამართავად.

-- ინსტალაცია

--- 1. რეპოზიტორიის კლონირება

```bash
git clone <repositoryurl>
```

--- 2. პროექტის დირექტორიაში გადასვლა

```bash
cd AI_Literacy_Project
```

--- 3. დამოკიდებულებების ინსტალაცია

```bash
npm install
```



-- გარემოს ცვლადები

პროექტის ძირეულ დირექტორიაში შექმენით `.env` ფაილი:

```env
DATABASE_URL=
NEXTAUTH_SECRET=
NEXTAUTH_URL=http://localhost:3000
```



-- მონაცემთა ბაზის კონფიგურაცია

Migrationების გაშვება:

```bash
npx prisma migrate deploy
```

Prisma Clientის გენერირება:

```bash
npx prisma generate
```



-- პროექტის გაშვება

განვითარების რეჟიმი:

```bash
npm run dev
```

აპლიკაცია ხელმისაწვდომი იქნება:

```text
http://localhost:3000
```

-- Branching Scenario მექანიზმი

სისტემაში გამოყენებულია Branching Scenario ტიპის შეფასება.

თითოეულ პასუხს შეუძლია მიუთითოს შემდეგ კითხვაზე `nextQuestionId` ველის გამოყენებით.

ეს უზრუნველყოფს:

* მრავალმიმართულებიან ქვიზებს;
* გადაწყვეტილებაზე დაფუძნებულ სწავლას;
* სხვადასხვა სასწავლო გზის შექმნას.
